## About

I build full-stack products and applied AI systems — end to end, not tutorials. My work ranges from a gamified learning platform serving real users to a transformer-based text classifier and a computer vision tool built on U²-Net. I care more about a project actually working, end to end, than about how many technologies it touches.

Day to day, I work across the **MERN stack** for full-stack products, **Python** for machine learning and computer vision, and **SQL** for the data layer beneath both. I'm early in my career and still learning — but everything below is a complete, working project, built by me, on purpose.

## Featured Projects

<br/>

### EduRights

An interactive learning platform that teaches children their fundamental rights through gamified modules — six learning modules, four interactive games, and real-time progress tracking with a points and badge system.

`React 19` `Vite` `Tailwind CSS` `Radix UI` `Node.js` `Express 5` `MongoDB`

**[Repository →](https://github.com/Sanikaa205/EduRights)** &nbsp;|&nbsp; **[Live Demo →](https://edurights.vercel.app)**

<br/>

### Depression Severity Analyzer

An AI-powered mental health support tool that classifies depression severity from user text using a fine-tuned RoBERTa transformer model, with confidence scoring, personalized coping suggestions, and curated video resources.

`Python` `FastAPI` `PyTorch` `Transformers (RoBERTa)` `React` `SQLite` `MongoDB`

**[Repository →](https://github.com/Sanikaa205/depression-prediction)**

<br/>

### Background Remover Pro

A computer vision application that removes and replaces image backgrounds using the U²-Net deep learning model, with support for static images, custom background compositing, real-time webcam virtual backgrounds, and MP4 video recording.

`Python` `Flask` `OpenCV` `Rembg (U²-Net)` `Pillow` `FFmpeg`

**[Repository →](https://github.com/Sanikaa205/Background-Remover)**
